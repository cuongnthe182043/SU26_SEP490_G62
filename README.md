# SU26_SEP490_G62

## Mobile app

Sau khi pull code về, cài dependencies cho mobile:

```powershell
npm install
```

Chạy Expo từ root repo:

```powershell
npm start
```

Nếu muốn cài lại riêng mobile:

```powershell
npm run setup
```

Hoặc chạy trực tiếp trong thư mục mobile:

```powershell
cd mobile
npm start
```
